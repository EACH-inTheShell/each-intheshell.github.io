ferramenta (linux, mac, windows)

Descrição:

john é uma ferramenta para craquear hashes em arquivos.

A ferramenta é um pouco enjoado quanto à formatação do arquivo de entrada, então para algumas das hashes que você achar, vai ter que "traduzi-las" para serem consumidas como entrada.

  • ssh2john - chaves SSH para o formato john
  • unshadow - junta os arquivos `/etc/passwd` e `/etc/shadow` no formato john

Uso:

john [opção]... <arquivo hash>

Opções:

  • --wordlist=<wordlist>:

    Define uma wordlist customizada para utilizar
