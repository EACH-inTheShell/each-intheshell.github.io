← hexedit

ferramenta (linux, mac)

Descrição:

Para o sistema operacional windows existe o Hexeditor↗

O hexedit é uma ferramenta para ver e editar arquivos em binários pelo terminal. Diferente de um bloco de notas normal é possível mexer com os bytes de um arquivo.

Na interface do hexedit o bloco da esquerda representa o byte inicial da linha, o bloco do meio representa os dados do arquivo em hexadecimal e o bloco da direita representa os dados do arquivo em ASCII (alguns dados não podem ser representados com caracteres ASCII, então eles são representados por pontos '.')

Uso:

  • Abrir o hexedit:

    • hexedit [opção] [arquivo]...

Opções:

  • -s/--sector: formata o programa em um tamanho fixo.
  • -m/--maximize: tenta maximizar o tamanho do programa.
  • -h/--help: mostra como usar.

CheatSheet:

  • F1:

    Abrir menu de ajuda

  • F2:

    Salvar o arquivo

  • F3:

    Carregar um arquivo

  • Setas:

    Se movimentar pelo arquivo

  • <, >:

    Ir para o começo e o fim do arquivo respectivamente

  • Return:

    Move para o endereõ especificado (em hexadecimal)

  • Ctrl-Z:

    Suspender

  • Ctrl-X:

    Salvar e sair

  • Ctrl-U:

    Desfazer todas as alterações

  • Crtl-S:

    Procurar por expressão à partir de onde o cursor está até o fim do arquivo

  • Ctrl-R:

    Procurar por expressão do inicio do arquivo até onde o cursor está
