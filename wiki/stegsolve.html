← stegsolve

ferramenta (linux, mac, windows)

Descrição:

Stegsolve é uma ferramenta feita em Java que permite a visualização e algumas transformações simples em imagem, extração de dados, resolver stereogramas, combinar imagens e verificar várias coisas como formatos de arquivo, bytes adicionais, buracos em arquivos etc... (dependendo do tipo de imagem)

Pode ser instalado aqui↗. Ou pode, caso exista, ser instalado através do pacote da ferramenta pelo seu gerenciador de pacotes.

As cinco funcionalidades podem ser encontradas na aba analyse.

Uso:

Caso tenha baixado o stegsolve.jar rodar:

java -jar stegsolve.jar

Caso tenha baixado pelo gerenciador de pacotes rodar:

stegsolve
