← steghide

ferramenta (linux, mac, windows)

Descrição:

Steghide é uma programa de que consegue esconder e recuperar informação dentro de imagens e arquivos de áudio.

Gambiarra para rodar steghide no macOS X AQUI↗.

Informação:

steghide info [-p] <arquivo>

  • -p <senha> (funciona nos 3 modos): senha que foi usara para embutir o arquivo.

Ver informações (metadados) sobre o arquivo embutido.

Extração:

steghide extract -sf <arquivo1> -xf <arquivo2> -p <senha>

O "arquivo1" é o arquivo que contém o arquivo embutido e o "arquivo2" é o arquivo que vai ser extraído.
  • -sf/--stegofile <arquivo>: recebe o arquivo que tem a informação imbutida.
  • -xf/--extractfile <arquivo>: recebe o nome arquivo que vai ser criado após a extração.

Embutir:

steghide embed -ef <arquivo1> -cf <arquivo2> -p <senha> [opções]

O "arquivo1" fica embutido dentro do "arquivo2".

Opções:

  • -ef/--embedfile <arquivo>: recebe como parâmetro o arquivo que vai ser embutido.
  • -cf/--coverfile <arquivo>: recebe como parâmetro o arquivo que vai ser usado para camuflagem. Arquivos suportados AU, BMP, JPEG ou WAV, pode ser encontrado no site↗.
  • -sf/--stegofile <arquivo>: recebe como parâmetro o nome do arquivo embutido.
  • -e/--encryption <algoritmo&gt <modo>: recebe dois parâmetros um algoritmo e um modo não importando a ordem em que vai ser recebido. Para ver os tipos de suportados usar o comando "steghide encinfo", usar -e sem nada para não encriptar. Por padrão é encriptado em rujndael-128 (AES) utilizando cbc
  • -z/--compress <1-9>: recebe como parâmetro um número de 1 a 9. Sendo 1 maior velocidade e 9 a melhor compressão.
  • -Z/--dontcompress: não comprime os dados
  • -K/--nochecksum: ele não embuteCRC32. Coloque se não quiser que ele gaste 32 bits a mais de memória ou já tem um cheksum.
  • -N/--dontembeedname: essa opção faz com que o não coloque o nome dentro do arquivo.

Exemplos:

Extrair:

Um arquivo da imagem "fotosecreta.jpeg" para "texto_super_secreto".

steghide extract -sf fotosecreta.jpeg -xf texto_super_secreto

Embutir:

Um arquivo "senha_super_secreta" em "sweater_weather.wav" com o nome "senhazinha".

steghide embed -ef senha_super_secreta -cf textosupersecreto -sf senhazinha

Um arquivo "nudes.jpg" em "garden.bmp" com criptografia rc2 no modo cbc

steghide embed -ef nudes.jpg -cf garden.bmp -e rc2 cbc
