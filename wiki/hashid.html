← hashid

ferramenta (linux, mac, windows)

Descrição:

O hashid é uma ferramenta feita em python3 para idenficar o tipo de hash que foi usado para criptografar o dado. Ele contém mais de 220 tipos de hashes guardados para comparação.

Uso:

hashid [opção...] <entrada>

A <entrada> pode ser tanto um hash escrito quanto vários arquivos em um diretório.

Opções:

  • -e <-extended>: lista todos os algoritmos de hash incluindo senhas com sal
  • -m <-mode>: mostra o modo correspondente do hash no hashcat
  • -j <-john>: mostra o modo correspondente do hash no JohnTheRipper
  • -o <-outfile FILE>: escreve a saída em um arquivo

Exemplos:

Verificando o hash "6a204bd89f3c8348afd5c77c717a097a"

hashid 6a204bd89f3c8348afd5c77c717a097a

Verificando todos os hashes dentro do arquivo "hash.txt" com o parâmetro -m para mostrar os modos correspondentes para o uso no hashcat

hashid -m hash.txt

Verificando todos os hashes de todos os arquivos da pasta atual com o parâmetro -j para mostrar os modos correspondentes para o uso no JohnTheRipper e colocando a sua saída em um arquivo "resultado.txt"

hashid -j ./* -o resultado.txt
